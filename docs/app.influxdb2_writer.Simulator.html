class Simulator(mosaik_api.Simulator):

View In Hierarchy

A mosaik data collector using an Influx database.

This mosaik simulator is able to take any number of inputs from mosaik and stores them in an Influx database.

The simulator expects to receive the simulation-internal time (as a ISO time string) on the `local_time` attribute and saves the data at the corresponding time in the database. Alternatively, a `start_date` can be provided which will be combined with the `step` and `time_resolution`.

This simulator will run as a `time-based` simulator when given a `step_size`, otherwise it will be `event-based`.
